# Homebrew Tap for Xanom

Desktop app for managing AI coding agents (Claude Code, Codex).

## Install

```sh
brew install neel-xanom/xanom/xanom
```

Or tap first, then install:

```sh
brew tap neel-xanom/xanom
brew install xanom
```

## Update

```sh
brew upgrade xanom
```

## Uninstall

```sh
brew uninstall xanom
```

To also remove all app data:

```sh
brew uninstall --zap xanom
```
